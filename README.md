# Web-приложение для детектирования и подсчета пицц

Данное приложение представляет собой простое веб-приложение на Python для детектирования и подсчёта пицц в видеопотоке с использованием предобученной нейросетевой модели YOLO в формате ONNX.

Приложение разработано в рамках учебной практики по теме применения искусственных нейронных сетей в задачах компьютерного зрения.

---

## Возможности

- загрузка ONNX-модели детектирования объектов;
- загрузка видеофайлов (MP4, AVI, MOV, MKV);
- настройка параметров инференса (confidence, IoU, размер входа);
- детектирование пицц на каждом кадре видео;
- визуализация bounding box’ов в режиме реального времени;
- подсчёт количества пицц в текущем кадре;
- сохранение размеченного видео;
- сбор статистики и истории запусков;
- экспорт истории в формате JSON и Excel.

---

## Используемые технологии

- Python 3.10+
- Streamlit
- OpenCV
- Ultralytics YOLO
- NumPy
- Pandas

## Установка и запуск

1. Клонировать репозиторий или скопировать файлы проекта.
2. Создать и активировать виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux / macOS
   venv\Scripts\activate     # Windows
   ```
3. Установить зависимости:
  ```bash
  pip install -r requirements.txt
  ```
4. Запустить веб-приложение:
```bash
python -m streamlit run app.py
```
5. Открыть браузер и перейти по адресу:
```bash
http://localhost:8501
```
